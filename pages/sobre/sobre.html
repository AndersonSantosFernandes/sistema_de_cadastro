Sobre projeto

  • Impede acesso direto pela URL, obrigando login para ter acesso
  • Validações de campos vazios
  • Validação de duplicidade de e-mail cadastrado
  • Valicação de CEP: formato correto ou inválido
  • Executa as quatro operações de CRUD
  • Gera um PDF a partir da lista de pessoas cadastradas
  • Resolve a formula de Bhaskara
  • Conversor de temperatura
  • Jogo de adivinhar o número sorteado
  • Sorteio randômico podendo escolher o intervalo

  • Tecnologias utilizadas no projeto

    • HTML
    • CSS
    • JavaScript
    • Bootstrapp
    • Sweet Alert

  • Utilizamos o localStorage como end point, onde podemos salvar as informações come se fosse um BD.
